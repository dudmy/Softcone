###기존 코드 심플하게 수정  
- 기존 소스에서 버튼 클릭 리스너 부분을 Switch문으로 단순하게 바꿈  
```java
@Override  
public void onClick(View v) {  
    switch (v.getId()) {  
        case R.id.btn_xxx:  
            break;  
        case R.id.btn_yyy:  
            break;   
    }  
}  
```
- 기존 소스의 여러 Activity를 한 개의 Activity와 여러 개의 Fragment로 나눔
